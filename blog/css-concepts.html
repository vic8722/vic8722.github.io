What does it mean to display inline vs using inline block?

Using the inline property for display will align the element

First, let's look at some definitions to give us somewhere to start. (Reference: w3schools.com)

  • block-level element - starts on a new line and takes up the full width available (stretches out to the left and right as far as it can.
  • inline element - does not start on a new line and only takes up as much width as necessary.

Hmm, ok. So an inline element makes sense. It doesn't start on a new line, which means it will display right next to the previous element in the flow.

Inline elements also will only take up as much space as necessary. That means we are not allowed to definie the size of the object, it will look at how much space is needed to fit an element of that type and place it in the browser, ignoring any size definitions.

Is that what you would have guessed it would look like? Since they still have the block aspect, they are still able to have their size defined, but are layed-out inline so they are next to one another on the same line!
